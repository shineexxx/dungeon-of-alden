# Подземелье Альдена

Консольный roguelike на Python с процедурной генерацией, биомами, магией, NPC, спецкомнатами и финальным боссом.

## Запуск

```bash
python3 game.py
```

Требуется терминал с поддержкой `curses` (Linux/macOS/Windows WSL).

## Управление

| Клавиша | Действие |
|---------|----------|
| `WASD` / стрелки | Перемещение |
| `G` | Подобрать предмет |
| `I` | Инвентарь / экипировка |
| `Z` | Кастовать заклинание |
| `B` | Книга заклинаний |
| `T` / `E` | Взаимодействие (NPC, алтари, сундуки...) |
| `J` | Журнал (бестиарий, лор, достижения) |
| `S` | Искать ловушки |
| `Пробел` / `Enter` | Спуститься по лестнице |
| `Esc` | Меню (сохранение, загрузка, новая игра, выход) |
| `Q` | Быстрый выход с автосейвом |

## Особенности

- **Процедурная генерация**: 6 разных биомов с уникальными генераторами, опасностями и ловушками.
- **Биомы**: каменное подземелье, пещеры, затопленный склеп, грибной лес, кузница ада, тёмные глубины.
- **NPC и торговля**: торговец, кузнец, алхимик, заклинатель, пленник — с диалогами и UI торговли.
- **Спецкомнаты**: алтари, фонтаны, сундуки, загадки, жертвенные круги, древние надписи.
- **Журнал**: бестиарий, опознанные предметы, посещённые места, собранный лор и достижения.
- **Магия**: заклинания с целями (self, single, line, cone, aoe), кулдаунами и маной.
- **Проклятия и опознание**: неизвестные зелья/свитки нужно опознавать.
- **Финальный этаж**: на 7-м уровне игрок встречает Тёмного Владыку.
- **Зал славы**: топ-10 результатов сохраняются между забегами.

## Структура проекта

```
content/        — данные: предметы, монстры, биомы, заклинания, события, лор
engine/         — игровой движок: ввод, рендеринг, цикл, меню, FOV, прицеливание
entities/       — игрок, мобы, NPC
systems/        — игровые системы: инвентарь, магия, прогрессия, эффекты, сохранения, журнал
world/          — генерация подземелья, биомы, ловушки, опасности, спецкомнаты, финал
tests/          — headless-тесты
```

## Сохранения

Сохранения хранятся в `saves/*.json`. Автосейв создаётся при спуске на новый уровень и при выходе (`Esc → Quit` или `Q`).

## Тестирование

```bash
for f in tests/test_*.py; do python3 "$f"; done
```
