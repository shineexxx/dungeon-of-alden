# Подземелье Альдена

Консольный roguelike на Python с процедурной генерацией, биомами, магией, NPC, спецкомнатами и финальным боссом.

## Запуск

```bash
python3 game.py
```

Требуется терминал с поддержкой `curses` и UTF-8 (Linux/macOS/Windows WSL). Минимальный размер окна: **60×14**.

## Управление

| Клавиша | Действие |
|---------|----------|
| `WASD` / стрелки | Перемещение |
| `G` | Подобрать предмет |
| `I` | Инвентарь / экипировка |
| `D` | Открыть инвентарь для выброса |
| `Z` | Применить заклинание |
| `B` | Книга заклинаний |
| `T` / `E` | Взаимодействовать с NPC или объектом |
| `J` | Журнал (бестиарий, лор, достижения) |
| `S` | Искать ловушки вокруг |
| `Пробел` / `Enter` | Спуститься по лестнице |
| `Esc` | Меню (сохранение, загрузка, настройки, новая игра, выход) |
| `Q` | Быстрый выход с автосейвом |

В меню/инвентаре: стрелки для выбора, `Enter` — подтвердить, `Esc` — назад.

## Интерфейс

### Верхняя строка (HUD)

```
☻30/30 ♦10/10 ⚔1 🛡0 ⭐1 XP:0/20 Гл:1 Каменное подземелье $:0 🍖:сыт
```

- `☻ HP/макс` — здоровье
- `♦ MP/макс` — мана
- `⚔ ATK` — урон
- `🛡 DEF` — защита
- `⭐ Lvl` — уровень
- `XP` — опыт до следующего уровня
- `Гл` — глубина и биом
- `$` — золото
- `🍖` — сытость

## Цель игры

Спуститься на 7-й уровень и победить финального босса — **Тёмного Владыка**. После победы результат попадает в Зал славы.

## Основные механики

### Бой
- Подходишь к монстру — атакуешь автоматически.
- Попадание зависит от броска `1d20` против защиты цели.
- Критический удар наносит двойной урон.
- Оружие может иметь бонусы: яд, огонь, вампиризм, заморозка.

### Инвентарь (`I`, `D`)
- Максимум 10 предметов в рюкзаке.
- Экипируй оружие, броню, кольца.
- Пей зелья, читай свитки, ешь еду.
- Неопознанные зелья и свитки имеют случайные описания до первого использования.
- **Выброс**: в инвентаре нажми `d`, чтобы выбросить предмет на пол под собой. Надетые предметы сначала снимаются (если не прокляты).

### Магия (`Z`, `B`)
- Заклинания тратят ману и могут иметь кулдаун.
- Типы целей: `self` (на себя), `single` (одна цель), `line` (линия), `cone` (конус), `aoe` (область).
- Книги заклинаний учат новым заклинаниям.

### Голод
- Сытость падает каждые несколько ходов.
- При сытости ≥ 50 медленно регенерируешь HP.
- При истощении (0) теряешь HP каждые ходы.

### Ловушки
- Скрытые ловушки активируются при наступании.
- `S` ищет ловушки в радиусе 1-2 клеток.
- Некоторые предметы/эффекты подсвечивают ловушки.

### NPC
- **Торговец** — покупает и продаёт предметы.
- **Кузнец** — улучшает оружие/броню за 100 золота.
- **Алхимик** — опознаёт все зелья в инвентаре.
- **Заклинатель** — продаёт книги заклинаний.
- **Пленник** — освободи его за награду (или предательство).

### Спецкомнаты
- Алтари дают бонусы или проклятия.
- Фонтаны исцеляют, восстанавливают ману или дают случайный эффект.
- Сундуки содержат лут, но иногда это мимик.
- Таблички с загадками, рычаги, жертвенные круги, древние надписи.

### Сохранения
- Автосейв при спуске на новый уровень и при выходе.
- Сохранения хранятся в `saves/` (не входят в git).
- В меню `Esc` можно вручную сохраниться/загрузиться.

### Настройки

В главном меню доступен пункт **Настройки**:

- **Графика**: `Unicode` (иконки, emoji-подобные символы) или `Классика` (ASCII-символы `# . > @ r g` и т.д.).

Настройки сохраняются в `saves/settings.json`.

### Зал славы
- Топ-10 результатов сохраняются в `saves/hall_of_fame.json`.
- Очки считаются по глубине, убийствам, золоту, уровню и победе.

## Советы для новичков

1. **Не спеши спускаться** — исследуй уровень, собирай зелья и еду.
2. **Всегда бери еду** — голод убивает не хуже монстров.
3. **Опознавай зелья** у алхимика или свитком опознания.
4. **Не открывай подозрительные сундуки без полного HP** — это может быть мимик.
5. **Ищи ловушки** (`S`) перед опасными участками.
6. **На 7-м уровне** готовься к боссу: полное HP/MP, хорошее оружие и броня.
7. **Кольца и артефакты** сильно меняют сборку — экспериментируй.
